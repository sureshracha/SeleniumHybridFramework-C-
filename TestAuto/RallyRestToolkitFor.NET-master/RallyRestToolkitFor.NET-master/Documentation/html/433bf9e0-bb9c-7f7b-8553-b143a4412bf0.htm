Parameters

isUiSupported
Type: SystemBoolean
Does this authentication manager support a UI?
applicationToken
Type: SystemString
An application token to be used as the file name to store data as (no extension needed). Each consuming application should use a unique name in order to ensure that the user credentials are not overwritten by other applications. An exception will be thrown elsewhere if this is not a valid file name.
encryptionKey
Type: SystemString
The encryption key, or salt, to be used for any encryption routines. This salt should be different for each user, and not the same for everyone consuming the same application. Only used for UI support.
encryptionRoutines
Type: Rally.RestApi.AuthIEncryptionRoutines
The encryption routines to use for encryption/decryption of data. Only used for UI support.
webServiceVersion (Optional)
Type: SystemString
The version of the WSAPI API to use.
